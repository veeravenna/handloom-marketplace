# Handloom Marketplace

A craft-first marketplace for authentic handloom products.

## About

This project is a modern web-based marketplace designed for artisan groups, cooperatives, and conscious buyers. It highlights traceable sourcing, craft storytelling, curated collections, and a premium marketplace feel while preserving the warmth of handmade textiles.

## Features

- Browse curated handloom collections
- Explore artisan stories and profiles
- Premium marketplace experience
- Responsive and modern UI design

## Tech Stack

- HTML5
- CSS3 (Custom properties, Flexbox, Grid)
- JavaScript (ES6+)

## Project Origin

This project was created using Perplexity AI as a concept design for a handloom-focused e-commerce platform.
